Connect your Nuki Smart Lock & Nuki Opener using the Nuki Bridge.

— Unlock or lock your Smart Lock with a Flow or the Homey app
— Create a Flow that starts when the door is locked or unlocked
— Create a Flow that starts when the battery is low
— Create a Flow that starts when the door is open

Please note that your Nuki Smart Lock must be first connected to Nuki Web before it can be found by Homey.
